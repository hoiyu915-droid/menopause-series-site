# 更年期系列網站

這是一個可直接部署到 GitHub Pages 的靜態網站，依照原始資料架構整理成「17 章文字 + 多張圖卡」。

## 檔案

- `index.html`：首頁、章節地圖、17 章內容、各章圖卡 gallery
- 圖卡放在 `cards/` 資料夾，檔名格式為 `01.01.PNG`
- 點擊圖卡可回到 Google Drive 原始圖卡
- 每章文字稿連回 Google Drive 原文

## GitHub Pages

把本資料夾內容放到 GitHub repo 的根目錄，在 GitHub Pages 設定中選擇 `main` branch / root 即可。

內容來源為 Google Drive 的「更年期」系列文稿與圖卡。
